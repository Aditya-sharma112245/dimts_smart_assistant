# DIMTS Smart Assistant

This repository contains a React + TypeScript chatbot component
developed for DIMTS internship submission.

## Description
- Client-side chatbot using React
- Keyword-based knowledge system
- No backend or external APIs required
- Designed for integration into the DIMTS website

## How to Use
- Import `DIMTSChatbot.tsx` into any React + TypeScript project
- Install required dependencies: react, lucide-react, tailwindcss
- Render the component using `<DIMTSChatbot />`

## Notes
AI tools were used as development assistants during implementation.
The chatbot logic and content were customized specifically for DIMTS.
